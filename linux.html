Terminal Kodlari

  • whoami : hangi kullanicida islem yaptigimizi gösterir.
  • pwd : oldugumuz klosörü gösterir.
  • hostname : baglanilan makinenin adini verir.
  • sudo su : root kullanicisina gecirir. sifre ister ve # olur.
  • clear : terminalde yazilmis olanlari temizler.
  • history : su ana kadar terminale yazilan komutlari gösterir.
  • history -c : gecmiste terminale yazilan komutlari siler.
  • ls : bulunulan dosyadaki dizinleri yan yana yazar.
  • ls -l : liste formatinda verir.
  • ls -la: gizli dosyalari liste halinde verir.
    Eger bir dosyanin basinda nokta(.) varsa o dosya gizli anlamina gelmektedir.
    Basinda d harfi olmasi o dosyanin bir klosör oldugunu ifade eder.
    rwx: read write execute demektir. Bu dizini olusturanin haklarini belirtir.
    3'lü sekilde gruplanir. -rwx-r-xr--
  • ls Desktop -la : alt klosörün icindeki ayrintilari girmeden gösterir.
  • cd ..: bir üst klosöre cikar
  • cd Desktop : yazilan klosöre girer.
  • ls /ho* : yarim yazilan ho ile baslayan tüm klosörleri getirir.
  • ls / : baslangic klosörüne gider.
  • uname : hangi programda calistigimizi gösterir.
  • uname -a : programin ayrintisini gösterir.
  • apt-get : paket manager i getirir.
  • ls /bin : bin klosörünü listeler. userlarin kullandigi binary ler burda bulunur.
  • ps : proc dosyasindan calisan prosessleri gösterir.
  • ls /proc/ : calisan proseslerin ayrintisini gösterir.
  • ls /proc/ -la : daha ayrintilica gösterir.
  • cd / : root(en üst) kullanicisina gider sembol ~ iken / döner.
  • cd ~ : home kullanicisina gider.
  • touch deneme.txt : istenilen formatta ve isimde dosya olustururuz.
  • mkdir klosor1 klosor 2 : iki adet klosör olusturur.
  • mkdir -p 1klosor/2klosor : klosör icinde klosor olusturur.
  • rm -rf klasor1 klosor2 : denilen klosörleri icinde dosya olsa da siler.

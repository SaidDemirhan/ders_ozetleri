Aritmetik Operatorler

  • += toplama islemidir.
    Eger bir degeri aciklama ile yazdirmak istersek arasina + veya , kaoyabiliriz.
    const kola=5;
    const cips=6;
    console.log("toplam fiyat",kola+cips);
    //toplam fiyat 11
    const sayi=7;
    const stringSayi="7";
    console.log(sayi+stringSayi);
    //77
  • - = cikarma islemidir.
  • * = carpma islemidir.
  • % = mod alma islemidir.
    const sayi=123;
    console.log(sayi%10);
    //3 birler basamgindaki degeri verir.
  • ++ = bir arttirma islemidir
  • -- = bir azaltma islemidir.
  • ** = us alma islemidir.
    const taban=2;
    const us=3;
    console.log(taban**us);
    //8(2*2*2)

NOT:Yazim sekli olarak;

Karsilastirma Operatorleri

  • == : esitligi kontrol eder
    const sayi=7;
    const stringSayi="7";
    console.log(sayi==stringSayi);
    //true
  • === : veri tipi de dahil olmak uzere esitligi kontrol eder.
    const sayi=7;
    const stringSayi="7";
    console.log(sayi==stringSayi);
    //false cünki biri integer digeri ise String bir deger.
  • != : esit olmama durumunu kontrol eder.
  • > : buyuk olup olmadigini kontrol eder
  • < : kucuk olup olmadigini kontrol eder
  • >= : esit veya buyuk olup olmadigini kontrol eder.
  • <= : esit veya kucuk olup olmadigini kontrol eder.

Mantiksal Operatorler

  • && : iki tarafin da istenilen sarti saglamasi gerekir
  • || : iki taraftan bir tanesi sarti sagliyorsa true verir.
  • ! : "degilse" manasina gelir.

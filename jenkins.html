JENKINS'İ CMD VEYA TERMİNAL İLE KURMA

  1. Bilgisayarınızda Jenkins dosyası oluşturun
  2. https://www.jenkins.io/download/ adresine gidip (Generic Java package(.war) Jenkins.war dosyasını, Jenkins klasörüne indirin
  3. CMD ekrenında Jenkins klasörünü açın ve java -jar jenkis.war ENTER
    KURULUM BAŞLAYACAKTIR.
    KURULUM İLERLEDİĞİNDE EKRANDA UZUN BİR ŞİFRE ÇIKACAK.
    O ŞİFREYİ KOPYALAYIN
  4. Browser adres satırına localhost:8080 yaz ENTER
  5. SİZDEN BİR ŞİFRE İSTEYECEK. AZ ÖNCE KOPYALADIĞINIZ ŞİFREYİ BURAYA YAPIŞTIRIN
  6. KURULUM DEVAM EDECEK. ŞİFRE VE KULLANICI ADI İSTEYECEK.
  7. Kurulum bittiğinde, Browser adres satırına localhost:8080 ENTER
  8. Username ve Password'u gir ENTER

JENKINS'İ ÇALIŞTIRABİLMEK İÇİN BAZI AYARLAR YAPMA

  1. Manage Jenkins => Global Tool Configration
  2. JDK => Instal automatically TIKLA
    Instaling JDK Oracle account. Please enter your username/password TIKLA
    Oracle username ve password'ünü gir.
  3. MAVEN => Maven installations TIKLA
    Maven versionu SEÇ
  4. Apply => Save

PROJE'Yİ JENKINS'TE ÇALIŞTIRMA

  1. New Item (Yeni Öğe) TIKLA
  2. Enter an item name = "isim ver"
  3. Freestyle projet'e TIKLA
  4. En Altta OK TIKLA
  5. General => GitHub project
    GitHub projenizin URL'ini buraya yazacaksınız
  6. Source Code Management (Kaynak Kodu Yönetimi) => Git TIKLA
    GitHub projenizin URL'ini buraya yazacaksınız
    Branch to build => Branch Specifier (blank for "any")
    */master yazısını */main olarak değiştir
  7. Build => Invoke top-level Maven targets SEÇ
    Goals => clean install YAZ
    Advance => POM => pom.xml YAZ
  8. Apply => Save
